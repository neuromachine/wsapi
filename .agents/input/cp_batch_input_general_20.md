# CP Batch Input

## Segment

```text
General digital services — web presence, e-commerce, booking, catalogs, corporate identity online.
Wide audience: SME, entrepreneurs, service providers, retail, B2B across industries.
```

## Geography

```text
Russia / CIS / Russian-speaking markets globally
```

## Target audience

```text
Business owners and entrepreneurs actively searching for a website, online store, or booking system.
Audience captured via interest-based internet advertising: "create a website", "online store turnkey",
"landing page for business", "corporate site order". Decision-makers at SME level.
```

## Output languages

```text
ru: required
en: optional
```

## WS positioning constraints

```text
WS is an engineering digital laboratory.
Focus on practical business systems: websites, booking, catalogs, CRM, automation, AI-assisted communication, analytics.
Do not sell "just a website". Sell a business tool.
```

## Pricing frame

```text
Currency: ₽ by default
Use realistic ranges.
Package 2 should be the recommended/balanced option.
```
